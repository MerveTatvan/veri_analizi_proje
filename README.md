# Öğrenci Notları Veri Analizi

Bu projede bir öğrenci veri seti kullanılarak temel veri analizi işlemleri yapılmıştır.

## Kullanılan Teknolojiler
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Yapılan Analizler
- CSV dosyasından veri okuma
- Bölümlere göre not ortalaması analizi
- Yaş ve not ilişkisi incelemesi
- Not dağılımının görselleştirilmesi

## Veri Seti
Veri seti `data/ogrenciler.csv` dosyasında bulunmaktadır.

## Sonuç
Bu çalışma, temel veri analizi sürecini ve görselleştirme adımlarını öğrenmek amacıyla hazırlanmıştır.
